# 部署说明书
## 每日报告中心 · Railway + Turso 部署指南

> 预计耗时：30 分钟  
> 难度：不需要任何技术背景，按步骤点击即可  
> 完成后你会得到：一个公开网址（如 `https://daily-report-xxx.railway.app`），发给同事直接访问

---

## 你需要准备的东西

- 一个电子邮箱（用来注册账号）
- 本项目的代码文件（就是这个 zip 包里的内容）
- GitHub 账号（没有的话第一步教你注册，免费的）

---

## 第一步：注册 GitHub 账号（代码存放的地方）

> GitHub 可以理解为"代码版的百度网盘"，用来存放网站代码。

1. 打开 [https://github.com](https://github.com)
2. 点右上角 **Sign up**，填邮箱、密码、用户名，完成注册
3. 登录后，点右上角 **+** 号 → **New repository**（新建仓库，相当于新建一个文件夹）
4. Repository name 填：`daily-report`
5. 选择 **Private**（私有，只有你能看到）
6. 点 **Create repository**

---

## 第二步：把代码上传到 GitHub

1. 解压你下载的 `daily-report.zip`，得到一个 `daily-report` 文件夹
2. 打开刚才创建的 GitHub 仓库页面
3. 点击 **uploading an existing file**（页面中间会有这个链接）
4. 把 `daily-report` 文件夹里的**所有文件和文件夹**拖进去（注意是文件夹里面的内容，不是文件夹本身）
5. 页面底部点 **Commit changes**（保存）

---

## 第三步：注册 Turso（数据库，存你的报告数据）

> Turso 就是一个"云端数据库"，专门用来存储抓取到的报告数据，免费套餐完全够用。

1. 打开 [https://turso.tech](https://turso.tech)，点 **Get Started Free**
2. 用 GitHub 账号直接登录（点 **Continue with GitHub**，更方便）
3. 登录后，点左侧 **Databases** → **Create Database**
4. 数据库名字填：`daily-report`，地区选 **Asia Pacific**（亚太，速度更快）
5. 点 **Create Database**

**获取连接信息（重要！后面要用到）：**

6. 点进刚创建的数据库，点上方 **Generate Token** 按钮
7. 会出现两行内容，分别复制保存到记事本里：
   - **Database URL**：格式像 `libsql://daily-report-xxx.turso.io`
   - **Auth Token**：一长串字母数字

---

## 第四步：注册 Railway 并部署（运行网站的地方）

> Railway 就是帮你"运行网站"的平台，你把代码给它，它帮你开着服务器，永远可以访问。

1. 打开 [https://railway.app](https://railway.app)，点 **Login** → **Login with GitHub**
2. 授权后，点 **New Project**（新建项目）
3. 选 **Deploy from GitHub repo**（从 GitHub 部署）
4. 选择你刚才创建的 `daily-report` 仓库
5. 点 **Deploy Now**，Railway 会开始自动安装和启动（等 1-2 分钟）

**配置环境变量（告诉网站数据库在哪里）：**

6. 部署完成后，点进这个项目，点上方 **Variables** 标签
7. 点 **New Variable**，添加以下两条（名字和值都要严格一致）：

| 变量名 | 值（填你第三步复制的内容）|
|--------|--------------------------|
| `TURSO_URL` | 你的 Database URL，如 `libsql://daily-report-xxx.turso.io` |
| `TURSO_TOKEN` | 你的 Auth Token |

8. 添加完后 Railway 会自动重新部署，等 1 分钟

**获取你的网址：**

9. 点 **Settings** → **Domains** → **Generate Domain**
10. 会出现一个网址，格式像 `daily-report-xxx.railway.app`，这就是你的网站地址！

---

## 第五步：访问并测试

1. 打开上面得到的网址，应该能看到「日报中心」界面
2. 点右上角 **立即抓取**，等待 2-3 分钟（第一次抓取要逐个访问来源网站）
3. 抓取完成后，页面会出现今日报告列表
4. 把网址发给同事，他们直接打开就能看到

## 常见问题

**Q：网站打开是空白或报错？**  
A：检查 Variables 里的 `TURSO_URL` 和 `TURSO_TOKEN` 是否填写正确，注意不要有多余的空格。

**Q：同事访问需要密码吗？**  
A：默认不需要密码，任何有网址的人都可以访问。如果你想加访问密码，告诉我，我帮你加一个简单的登录页。

**Q：每天自动抓取几点运行？**  
A：每天北京时间早上 8:00 自动运行一次，你也可以随时手动点「立即抓取」。

**Q：Railway 会收费吗？**  
A：免费套餐每月有 500 小时运行时间，一个项目每月 31 天 × 24 小时 = 744 小时，超出部分很便宜（约 $5/月）。如果想一直免费，可以升级到 Hobby 计划（$5/月 含 $5 额度，基本够用）。

**Q：我想换一个自己的域名（如 `report.mycompany.com`）？**  
A：在 Railway → Settings → Domains 里点 **Custom Domain**，按提示操作即可。
